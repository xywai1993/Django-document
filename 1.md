# 1.3 如何组织文档

Django 有许多文档。一份高度概览的文档会帮助你知道从哪里查找正确的信息：

-   教程使你通过一系列的步骤亲手创建一个应用程序。假如你是第一次使用 Django 或者第一次开发 web 应用请从这里开始。观看“第一步”
